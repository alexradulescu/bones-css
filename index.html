Bones CSS

Use it
        
          Simple, Beautiful, Lightweight, CSS boilerplate for prototypes

Simple

Styling plain html elements. No css classes to remember

Beautiful

Did my best to make it look good. Suggestions are still welcome though ðŸ˜‰

Lightweight

Less than 600 lines of code (excluding comments)

Great for

Quick prototypes or tutorials, locally or on codesandbox, gitpod, stackblitz,

Not good for

Serious or big projects because it targets html elements directly. Also it can override other css frameworks and there are many things missing which can be done only with classes

Inspired by

Skeleton css, Water css, Tailwind css, 1-line layouts

How to use

Copy&Paste the following code in the <head> of your index.html 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@alex.radulescu/bones-css@0.9.0/dist/bones.77e59c7f.css" />

If you like Bones, feel free to Star it or raise an issue or a suggestion here Issue

Forms

Forms are a huge pain, but hopefully these styles make it a bit easier. All inputs, select, and buttons are normalized for a common height cross-browser so inputs can be stacked or placed alongside each other.

Solitary inputs (not in a form or label) get a default standardised size. 

            <input type="search" placeholder="Search..." /> <button>Find</button>

Input elements within a form or fieldset get a full width
