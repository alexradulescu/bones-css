Bones CSS
Use it
        
          
            Simple, nice, lightweight, css boilerplate for prototypes
Simple

No css classes to remember at all. Just well structured plain html.

Nice

Did my best to make it look good. Suggestions are still welcome ðŸ˜‰

Lightweight

Less than 600 lines of code (excluding comments).

Great for

Quick prototypes or tutorials, locally or on codesandbox, gitpod, stackblitz,

Not good for

Serious or big projects because it targets html elements directly. Also it can override other css frameworks and there are many things missing which can be done only with classes

Inspired by

Skeleton css, Water css, MVP css, 1-line layouts, Tailwind css, Material design

How to use

Copy&Paste the following code in the <head> of your index.html 


<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@alex.radulescu/bones-css@0.6.1/dist/bones.aac0285f.css" />
